<h1>University Management System</h1>

Простой CRUD-сервис для управления списком студентов. Написан на чистом PHP в учебных целях, чтобы отработать навыки объектно-ориентированного программирования и паттерна MVC без использования тяжелых фреймворков.

<h5>Что внутри:</h5>

<ul>

<li>Архитектура: Самописный MVC (Model-View-Controller).</li>
<li>Бэкенд: PHP 8.x (OOP), работа с базой через PDO.</li>
<li>База данных: MySQL.</li>
<li>Фронтенд: Минималистичный интерфейс на Bootstrap.</li>

</ul>


<h5>Функционал:</h5>

<ul>
<li>Авторизация администратора (проверка сессий).</li>
<li>Полный цикл CRUD для работы со студентами (создание, чтение, обновление, удаление).</li>
<li>Прямое управление админами через БД (для упрощения логики проекта).</li>
</ul>

<h5>Установка:</h5>

В корне папке этого проекта есть файл, который называется oop_project_2.sql. 
<br>В phpMyAdmin создайте БД и назовите ее "oop_project_2" и импортируйте этот самый файл.
<br><strong>Важно!</strong> другое название использовать нежелательно, так как возникнут проблемы с подключением.
<br>(Это будет исправлено в следующих апдейтах). 

<br>Проверьте настройки подключения к БД в файле /config/database.php, где создается объект new Database(); 
<br>Если у вас другой пароль от MySQL или порт, просто поправьте эти параметры под себя.

<br>Откройте проект через OpenServer, MAMP или любой другой локальный сервер.
