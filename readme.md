<h1>University Management System</h1>

Простой CRUD-сервис для управления списком студентов. Написан на чистом PHP в учебных целях, чтобы отработать навыки объектно-ориентированного программирования и паттерна MVC без использования фреймворков.

<h5>Что внутри:</h5>

<ul>

<li>Архитектура: Самописный MVC (Model-View-Controller).</li>
<li>Бэкенд: PHP 8.x (OOP), работа с базой через PDO.</li>
<li>База данных: MySQL.</li>
<li>Фронтенд: Минималистичный интерфейс на Bootstrap.</li>

</ul>


<h5>Функционал:</h5>

<ul>
<li>Авторизация администратора (проверка сессий).</li>
<li>Полный цикл CRUD для работы со студентами (создание, чтение, обновление, удаление).</li>
<li>Прямое управление админами через БД (для упрощения логики проекта).</li>
</ul>


<h5>Примечание: Форма регистрации администраторов пока отсутствует. Все администраторы добавляются напрямую через phpMyAdmin. При создании администратора важно указать уровень его прав в поле <strong>role (int)</strong>. Всего доступно 4 уровня администраторов. По умолчанию устанавливается уровень <strong>0</strong>. Обратите внимание: с нулевым уровнем доступа вы не сможете войти в Dashboard.</h5>

<ul>
<li>
 <strong>1 уровень: Младшие администраторы</strong>
 <p>
 Права: просмотр списка студентов (вкладка View).
 <br>
 В контексте проекта данный уровень назначается преподавателям.
 </p>
</li>

<li>
 <strong>2 уровень: Администраторы</strong>
 <p>
 Права: просмотр списка студентов (вкладка View) и редактирование существующих записей студентов.
 <br>
 В контексте проекта данный уровень назначается заместителю директора учебного заведения.
 </p>
</li>

<li>
 <strong>3 уровень: Супер-администраторы</strong>
 <p>
 Данный уровень предоставляет полный доступ ко всем действиям со студентами: добавление, редактирование, просмотр и удаление.
 <br>
 В контексте проекта этот уровень назначается председателю учебного заведения.
 </p>
</li>

<li>
 <strong>4 уровень: Особые администраторы</strong>
 <p>
 Данный уровень позволяет просматривать список студентов и исключать (удалять) их из учебного заведения.
 <br>
 В контексте проекта этот уровень назначается президенту ученического совета.
 </p>
</li>
</ul>

<h5>Установка:</h5>

В корне папке этого проекта есть файл, который называется oop_project_2.sql. 
<br>В phpMyAdmin создайте БД и импортируйте этот самый файл.

<br>Проверьте настройки подключения к БД в файле /config/db_config.php 
и подкорректируйте поля(servername, username, password, database) под себя

<br>Откройте проект через OpenServer, MAMP или любой другой локальный сервер.

<h1>Проект находится в стадии разработки и доступен в раннем доступе.</h1>
